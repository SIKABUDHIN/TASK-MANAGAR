# **App Name**: TaskFlow

## Core Features:

- Task Creation: Allow users to create tasks with title, description, priority, due date, and status using a modal form.
- Task Categorization: Categorize tasks into 'To-Do', 'In-Progress', and 'Completed' boards.
- Drag and Drop: Enable users to drag and drop tasks between boards to update their status.
- Task Editing: Allow users to edit task details and status by clicking on a task.
- Task Deletion: Provide an option to delete tasks with a confirmation step.
- Filtering and Sorting: Implement filters for priority, due date, and status, and sorting options for oldest, newest, and closest due date.
- Data Persistence: Load initial tasks from a JSON file and persist all updates using localStorage.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to provide a calm and organized feel, inspired by productivity and clear skies.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce visual clutter.
- Accent color: Muted green (#A7D1AB) to highlight important actions and provide a sense of progress.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look.
- Use simple, minimalist icons to represent task categories and actions.
- Implement a responsive 3-column board layout that adapts to different screen sizes.
- Subtle animations when dragging and dropping tasks between boards to provide visual feedback.
